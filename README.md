# PruebaIntegrity
Esta es una prueba para puesto de trabajo en Integrity

## Descripcion de la prueba

Realizar Monitor de finanzas utilizando solamente NodeJS, JavaScript, HTML Y CSS vanilla sin ningun framework ni modulos externos.

## Ejecutar la app

Para ejecucion del servidor situarse en la raiz del proyecto backend y ejecutar el comando:
 ```
 > npm run start
 ```
Para visualizacion del frontend: 
 ```
 Abrir el archivo index.html del proyecto frontend en el navegador
 ```

Una vez realizado las dos indicaciones se podra acceder al localhost para el servidor del backend en el puerto 3000.
